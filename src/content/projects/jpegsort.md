**<a href="https://github.com/nikokomninos/JPEGSort" target="_blank" class="animate-gradient">JPEGSort</a>** is a simple
python script to sort and copy jpg files into directories by year, based on EXIF metadata, using 
the Pillow library for Python. Initially built to sort some family photos.
